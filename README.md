# System Zarządzania Wynajmem

## Cel
Aplikacja konsolowa do zarządzania danymi o najemcach mieszkań.

## Funkcjonalności
- Wyświetlanie danych najemców,
- Dodawanie najemców,
- Edycja czynszu,
- Sortowanie wg czynszu,
- Zapisywanie danych do plików JSON.

## Autorzy
- Szymon Karbownik
- Patryk Jurek

## Uruchomienie
```
python main.py
```

## Przykład danych
```json
[
  {
    "name": "Jan",
    "surname": "Kowalski",
    "room": {
      "number": "1A",
      "rent": 1500
    }
  }
]
```
